## About project

Twin Peaks TV is a cross-platform smart TV application developed using the Flutter framework.
The project demonstrates the capabilities of the
[tv_plus](https://github.com/dinaraparanid/tv_plus) library
on 4 smart TV platforms: Android TV, Apple TV, LG webOS and Samsung Tizen.
